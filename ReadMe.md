##### CÀI ĐẶT DJANGO 

* ###### tao moi truong ao 

&nbsp;	- Mở PowerShell bằng quyền Administrator

&nbsp;		- Set-ExecutionPolicy RemoteSigned

&nbsp;		- Khi được hỏi: Do you want to change the execution policy? (Y/N) chọn Y

&nbsp;	- Mo terminal tren vs code (ctrl + ~)

&nbsp;		chạy lệnh 

&nbsp;		win: venv\\Scripts\\activate 

&nbsp;		mac: source venv/bin/activate



* ###### tai Django

&nbsp;	- pip install Django

* ###### tao account admin

&nbsp;	- python manage.py migrate

&nbsp;	- python manage.py createsuperuser

&nbsp;	- python manage.py runserver

* ###### vo trang quan ly user

&nbsp;	mo web va chay link:

&nbsp;	- http://127.0.0.1:8000/admin

&nbsp;	nhap tai khoan vua moi tao de dang nhap
